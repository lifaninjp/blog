Insert Error And Encoding

  1. Mysql Command Line

Make sure all charset have been set to utf8.

  1. PhpMyAdmin
    Set database,table and column to utf8-general-ci.

  2. php.ini
    Set encoding and mbstring.encoding to UTF-8 in php.ini.
